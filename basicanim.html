Theme 04: Basic Animation


The purpose of this module was to learn how to be able to design and build a game on a website that has visually engaging characters and UI elements with CSS animations and manipulatig the input flow using JavaScript.

A website with a Game

Description

04.01.01 Individual assignment

Process & Knowledge

The game is a single-screen clicker game, where a user can click on good or bad objects. When you click on them, you gain points and lose lives. The game is won when the user clicks on 25 good items within a 1-minute time frame.The game is lost when all 3 lives are lost by clicking on bad items or the timer runs out.

Technically, the game is not very impressive, as I've used the same set of animations and functions that were taught in class because I spent a bit too much time on the aesthetics.I would have wanted to have the items be more interactive with the character in a more engaging way. In the future, a personal project.
